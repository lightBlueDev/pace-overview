# Examples

This directory contains sanitized, documentation-oriented examples that illustrate the shape of PACE artifacts without exposing private project materials.

Included examples:

- `sample-project-doc-stack.md`: role-separated document stack for a project
- `sample-task-slice.md`: atomic execution unit with acceptance criteria and closure signals
- `sample-closure-checklist.md`: multi-surface completion standard

These examples are intended to make the operating model concrete for reviewers and collaborators.
